## 1. 本 note 的目的

本 note 用来修正 `Track T0` 的事实源判断。

先前把 `丽花义正-交易谱（1975.01-1976.12）` 目录下的月度 pdf 当作原始交易事实页，这是不准确的。经用户澄清和后续核对，当前更可信的事实源应改为：

1. 台湾版图片页 `B0288-B0308`
2. 两本整书后几十页中的原始月度表

## 3. 当前最重要的新发现

后页月度表揭示出一个比此前更准确的结构：

1. `买 / 卖` 不是一列净额，而是双向栏位
2. `未平仓` 也不是单值，而是 `多 / 空` 双栏
3. 这意味着立花在这些样本里确实存在 `锁单共存` 语义

因此后续数据契约不能再只写：

1. `action`
2. `open_units`

而应至少区分：

1. `buy_units`
2. `sell_units`
3. `open_long_units`
4. `open_short_units`

---

## 4. 当前结论

`Track T0` 的真正事实源，当前应从“自建月度 pdf / xlsx 骨架”迁移为“书后页月度表 + 用户自建骨架联合校对”。`

也就是说：

1. 用户自建月度表负责骨架与对齐
2. 书后页负责真实交易动作与未平仓事实
