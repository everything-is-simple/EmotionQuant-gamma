## 1. 文档定位

本文档是 `Normandy` 战役专属的**方法论 dossier 与理论摘要**，不是共享 `Strategy` 来源文，也不是当前主线设计文。

它保留的价值主要是：

1. 记录立花义正与 `YTC / Volman` 明显不同的一条交易思路
2. 说明这套方法为什么不能被粗暴理解成“把 PAS 反着做”
3. 为未来若要研究极端情绪、逆向博弈与执行纪律，保留一个真实起点

当前主线中，立花义正方法**不构成** `PAS` 或 `MSS` 的正式实现来源，也不进入当前交易链路的正式契约。

从 `2026-03-12` 起，它正式归入 `Normandy` 研究资产层，不再保留在共享 `docs/Strategy/`。

---

## 2. 为什么保留，但不直接采用

立花义正材料值得保留，是因为它提醒了三件事：

1. 市场极端一致时，最值得看的往往不是“更顺着做”，而是“一致预期何时开始失效”
2. 交易体系除了形态识别，还存在一整套与 `试探 / 加减 / 撤退 / 休息` 有关的执行纪律
3. 逆向体系不是顺势体系的反面涂层，而是另一条独立的行为链

但它不适合直接进入当前主线，原因也同样明确：

1. 当前仓库能直接复核的原始资料仍然以书稿、整理稿和图像版为主，缺少可直接 formalize 的统一参数表
2. 它更偏交易哲学、执行纪律与主观博弈，不适合直接翻译成当前 `A股日线 + T+1 Open` 的 formal 契约
3. 当前主线已经把 `PAS` 固定为个股形态层，把 `MSS` 固定为市场级风险调节，不宜再引入一套跨层逆向覆盖逻辑

## 3. 当前可稳定提炼出的理论骨架

下面这些内容，不是对全部原著的逐章复写，而是结合仓库现有整理稿与原始资料目录回看后，当前最稳妥、最不容易失真的理论骨架。

### 3.1 它先看“大众是否过度一致”，再看“一致何时失效”

立花义正方法最有辨识度的一点，不是“逢反向就做”，而是：

1. 先识别市场大众是否已经明显站到一边
2. 再观察这股一致力量是否开始露出疲态
3. 真正有价值的时点，不是情绪本身，而是情绪开始失效的那一下

因此它关注的不是：

`单纯和别人反着来`

而更像：

`大众共识过满之后，谁先被迫撤退`

### 3.2 它不把“预测方向”放在第一位，而把“试探与修正”放在第一位

这套方法的另一条核心，不是一次性做对，而是：

1. 先用小仓位试探
2. 证据转强再加码
3. 证据不成立就撤退

也就是说，它把交易更像看成：

`持续修正的操作过程`

而不是：

`开仓前一次性完成的预测判断`

这一点和当前主线 `PAS` 的区别非常大。  
`PAS` 更像先确认形态再发单；立花义正更像先试探，再让仓位随着证据移动。

### 3.3 它强调分批、试单、加减仓，而不是一次性重仓表态

从仓库现有整理稿能稳定回收出的一个高频主题是：

1. 不轻易一次性压满
2. 分批进入、分批处理
3. 让仓位和市场反馈同步变化

这条思路的价值不在于“分三次”这个数字本身，而在于：

`仓位是被市场验证出来的，不是被信念一次性决定的`

如果未来真的研究这条线，更值得 formalize 的不是“固定三分之一”，而是：

1. `test size`
2. `confirmation add`
3. `failure retreat`

### 3.4 它比一般形态书更强调“撤退、休息、避免硬扛”

立花义正方法和很多只讲入场的人不同，它明显更重视：

1. 错了就撤
2. 状态不好就休息
3. 不把一次交易失误硬扛成长期损耗

这说明它的重点并不只是“抓反转”，而是：

`如何在错误判断出现时，把伤害尽快收口`

这也是为什么它对当前仓库有价值：  
它提醒我们，交易体系不只有 entry alpha，还有执行节奏和心理耐久。

### 3.5 它更像“行为失衡 doctrine”，不是“价格形态 taxonomy”

`YTC / Volman / 当前 PAS` 都更偏：

1. 结构
2. 形态
3. 测试 / 回撤 / 假突破 / continuation

而立花义正更偏：

1. 大众情绪
2. 试探与撤退
3. 错误方向持仓何时开始自我瓦解
4. 交易者自己如何避免被市场反向收割

所以它和 `YTC` 的差别不只是“顺势 vs 逆势”，而是：

`一个更像形态语言，一个更像执行与行为语言`

## 4. 它和当前 PAS / Volman / YTC 的真正关系

### 4.1 它不是 PAS 五形态的新补丁

当前 `PAS` 的 taxonomy 已固定为：

1. `BOF`
2. `BPB`
3. `PB`
4. `TST`
5. `CPB`

立花义正方法不属于这张表，也不该被硬塞成第六、第七种形态。

### 4.2 它也不是把 PAS 反着做

更合理的理解是：

1. `PAS` 更偏结构确认后的顺势形态
2. 立花义正更偏极端一致后的逆向博弈
3. 两者既不对称，也不互为镜像反信号

因此最需要避免的误读是：

`PAS 做多，立花就做空`

这不是它真正的价值。

### 4.3 它和 Volman 的分歧点尤其明显

`Volman` 重视的是：

1. pressure
2. handle
3. breakout / pullback / second chance
4. 结构推进中的 trapped traders

立花义正更重视的是：

1. 大众情绪何时极端
2. 自己如何先小仓试探
3. 错了如何马上退
4. 做完一段之后如何停手

一句话压缩：

`Volman 更像结构交易员的语言，立花义正更像执行型逆向交易者的语言。`

## 5. 当前保留结论

本文件当前固定保留下面五条结论：

1. 它是一个有价值的逆向交易理论样本
2. 它最有价值的部分，不是“反着做”，而是“识别大众一致何时失效”
3. 它对当前仓库的启发主要落在：极端情绪观察、试探仓位、撤退纪律、休息纪律
4. 它不作为当前主线 `PAS` 或 `MSS` 的直接来源文档
5. 它不承担任何实现计划、参数设计或测试验收职责

## 6. 更合理的未来研究落点

如果未来真的要吸收这条思想，更合理的落点不是直接改写当前主线，而是开一条受控研究线。

当前最合理的三个研究方向是：

1. `TACHI_CROWD_FAILURE`
   - 目标：把“极端一致 -> 失效 -> reclaim”收成第一条最小可执行 entry 假设
2. `Tachibana contrary theory dossier`
   - 目标：把“极端一致 -> 失效 -> 试探 -> 加减 -> 撤退”整理成明确理论链
3. `extreme sentiment sample study`
   - 目标：在 A 股里找出最接近“大众一致开始失效”的样本族
4. `execution rhythm discipline study`
   - 目标：研究 `试单 / 减仓 / 休息 / 停手` 是否能形成独立的执行纪律层

这里最关键的边界是：

1. 不把它直接并入当前 `PAS registry`
2. 不把它直接翻译成 `MSS` 新状态机
3. 先做研究样本和理论 dossier，再谈是否值得 formalize
