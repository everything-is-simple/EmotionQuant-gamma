## 1. 这张卡回答什么

这张卡当前只回答一个问题：

`在 Phase 9B 已经出现多个 isolated winners 之后，Phase 9C 现在到底允许哪几个组合候选进入后续验证，以及哪些字段必须继续排除在外。`

截至 `2026-03-19`，当前真实状态已经是：

1. `GX8 / three-level trend hierarchy` 已完成，不再阻塞组合 scope
2. `Phase 9B` 已产生 `3` 个 isolated winners
3. `17.6 / Phase 9C` 才是当前真实下一步

因此这张卡当前先做的是：

`formal combination freeze`

不是：

1. 直接跑组合回放
2. 再新开新的 Gene 定义卡
3. 提前进入 `17.7 / Phase 9D package promotion ruling`

## 3. Frozen Source Field Pool

当前只允许从下面 `3` 个已经赢下 isolated round 的字段里选组合候选：

1. `duration_percentile`
2. `reversal_state`
3. `context_trend_direction_before`

单字段结论在本卡中的角色固定为：

`upstream isolated evidence only`

它们不是本卡要重新枚举的 Phase 9C 组合候选集。

## 4. Frozen Allowed Combination Set

当前 Phase 9C 明确允许的组合候选只有 `4` 个：

1. `duration_percentile + reversal_state`
2. `duration_percentile + context_trend_direction_before`
3. `reversal_state + context_trend_direction_before`
4. `duration_percentile + reversal_state + context_trend_direction_before`

除这 `4` 个之外，本卡不允许：

1. 把单字段重新伪装成“组合候选”
2. 临时加第四个字段
3. 用隐藏附加条件、解释性补丁、排序补偿或 tie-break 偷带额外变量

## 5. Explicitly Forbidden

当前明确禁止偷带以下字段：

1. `wave_role`
2. `current_wave_age_band`
3. `mirror`
4. `conditioning`
5. `gene_score`

这些禁止项在本卡内同时适用于：

1. 组合定义
2. 候选排序
3. tie-break
4. 补充 gate
5. 解释性“顺手加一条”

## 6. Boundaries

本卡当前明确不做下面这些事：

1. 不在本卡里扩写 runtime / SQL / 脚本字段映射
2. 不在本卡里发明 `reversal_state` 或 `context_trend_direction_before` 的新别名
3. 不在本卡里定义 package promotion / retention / defer 结论
4. 不在 freeze 之外口头授权额外组合 replay
