# Phase 9E / duration_percentile 书义寿命分布重开卡

- 状态: `Active`
- 日期: `2026-03-19`
- 类型: `single-variable follow-up validation`
- 范围: `按书义重开 duration 轴，把中级主要走势的连续寿命分布与四分位统计先做对，再决定 runtime 是否值得继续`

## 1. 这张卡回答什么

这张卡只回答一个问题：

`如果按《专业投机原理》的原义，把寿命轴收回到“长期趋势里的中级主要走势”，并用波段时间 + 波动幅度的连续分布和四分位来读当前阶段，那么 duration 轴还值不值得继续作为 Phase 9 的 runtime 候选？`

---

## 2. 为什么必须开这张卡

当前仓库历史上只留下了：

1. `p95` formal round
2. `p65` sensitivity reference

但你刚刚重新钉死了书义目标：

1. 长期趋势 = 我们俗称的 `牛市 / 熊市`
2. 真正要统计的不是整轮长期牛熊本身，而是：
   `长期趋势里，夹在两个次级折返走势之间的中级主要走势`
3. 统计图不该把眼光钉在 `p65 ~ p95` 这种尾部老年带
4. 应回到：
   `first-1/4 / second-1/4 / third-1/4 / fourth-1/4`
   也就是 `0~25 / 25~50 / 50~75 / 75~100`
5. 这张图必须同时消费：
   - `波段时间`
   - `波动幅度`
   并形成连续分布

所以旧问题已经不再是“p65 ~ p95 之间哪个点最好”，而是：

1. 旧 `p95 / p65` isolated reading 还能不能保留为历史 round
2. 新寿命图在四分位连续分布下，当前样本主要落在哪个阶段
3. 哪个四分位区间如果未来要进入 runtime，才算有书义基础
4. `duration_percentile` 是继续当 runtime candidate，还是应该退回 sidecar distribution only

另外，`2026-03-19` 的 `GX12` 已正式裁定：

1. `GX10` 已改变 duration 轴的参考基础
2. 因此本卡不能继续沿整改前 duration surface 往下跑
3. 当前任何未完成的 sweep 尝试，都应视为：
   `pause legacy-surface attempt and reopen on remediated duration surface`

## 4. Distribution Boundary

本卡仍然只允许改变一个主变量：

`duration / lifespan surface`

但它当前不再允许继续用旧的尾部 threshold 轴推进：

1. 不再以 `p65 / p70 / p75 / p80 / p85 / p90 / p95` 作为正式主轴
2. 不再默认把 `block when current_wave_duration_percentile >= threshold` 当成唯一 truthful rule shape

当前新的正式研究边界固定为：

1. 样本对象优先锁到：
   `INTERMEDIATE + MAINSTREAM + long-context-consistent`
2. 分布读法改为：
   `FIRST_QUARTER / SECOND_QUARTER / THIRD_QUARTER / FOURTH_QUARTER`
3. 连续分布图必须同时保留：
   - `duration`
   - `magnitude`
   - `lifespan_joint_percentile`
4. 只有在这张分布图跑出来之后，才允许问：
   - 某个四分位是否值得做 runtime negative guard
   - 或 duration 应彻底退回 sidecar only

本卡可以研究：

1. 四分位密度分布
2. 连续分布拐点
3. 宽度与持续期是否共振
4. 联合寿命分位与回测收益分布的关系

但本卡**不能**直接跳成：

1. 先验认定 `FOURTH_QUARTER` 就该全部挡掉
2. 把 `p65 / p95` 换个名字继续当正式主轴
3. 还没重跑完寿命分布就直接开 `17.9` 组合 replay

1. 一份 remediated lifespan distribution evidence JSON

   - `duration quartile sample counts`

6. 一个明确结论：
   - `duration remains a truthful runtime candidate under quartile surface`
   - 或 `duration should return to sidecar-only distribution reading`
