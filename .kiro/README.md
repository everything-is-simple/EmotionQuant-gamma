# .kiro 目录导航

## 目录定位

`.kiro/` 是执行期工作区，存放“当前开发过程中的记录与工作副本”。

## 子目录职责

1. `roadmap/`
   - 当前执行阶段工作副本（按版本子目录组织）。
   - 仅当前激活阶段持续更新，其余阶段保留占位目录。
2. `record/`
   - 开发过程记录（状态、可复用资产、技术债等）。
3. `steering/`
   - 执行规则与工程约束（流程、约定、A 股规则等）。

## 与 docs 的关系

1. `docs/spec/` 是阶段文档“全量归档”。
2. `.kiro/roadmap/` 是当前执行阶段“工作副本”。
3. 版本切换时，先更新 `.kiro/roadmap/README.md` 的 Active 阶段说明。
