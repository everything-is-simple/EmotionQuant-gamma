# v0.01 工作集索引

## 目录定位

本目录是 **当前正在实现** 的 `v0.01` 工作集。  
你在实现阶段只需要优先看这里，不用在多个目录来回找。

5. `v0.01-mvp-spec-04-broker.md`  
   - Broker 模块实现卡。
6. `v0.01-mvp-spec-05-backtest-report.md`  
   - Backtest/Report 模块实现卡。
7. `v0.01-boundary-deterministic-rules.md`  
   - 边界条件与确定性规则（停牌/涨跌停/订单状态）。
8. `v0.01-data-contract-table.md`
