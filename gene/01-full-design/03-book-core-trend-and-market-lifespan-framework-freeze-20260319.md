## 3. 当前冻结的核心定义

### 3.1 趋势不是指标，先是结构

趋势的第一性定义固定为：

1. 上升趋势：
   `一系列依次抬高的高点与低点`
2. 下降趋势：
   `一系列依次降低的低点与高点`

这意味着：

1. Gene 不能先从评分器出发，再倒推趋势
2. Gene 必须先切出结构对象，再谈 percentiles / z-score / score

### 3.2 三层趋势必须并存

本轮固定采用三层趋势并存语义：

1. `LONG`
   - 长期趋势
   - 对应市场俗称 `牛市 / 熊市`
2. `INTERMEDIATE`
   - 中级趋势
   - 对应长期趋势内部的重要中期走势
3. `SHORT`
   - 短期趋势
   - 对应更短的摆动与噪音层

Gene 后续所有“当前趋势”说法，都必须显式带层级。

### 3.3 真正要统计的对象不是整轮牛熊，而是中级主要走势

从 `159 ~ 167` 页开始，当前冻结把寿命统计对象正式写定为：

1. 长期趋势：
   `多头市场 / 空头市场`
2. 中级主要走势：
   `长期趋势内部，夹在两个次级折返走势之间的那段主要走势`
3. 中级次级修正走势：
   `与主要趋势相反的重要中期走势`

因此 Gene 的 canonical lifespan object 不是：

1. 整轮长期牛熊
2. 任意方向 completed wave
3. 混入次级折返的全部中级波段

而是：

`长期趋势内部的中级主要走势`

### 3.4 趋势改变不是一句话，而是一组确认法

基于 `68 ~ 85` 页，当前冻结把趋势改变的确认法写定为四层：

1. 趋势线被突破
2. 上升趋势不再创新高，或下降趋势不再创新低
3. 反向跌破/上破前一个重要短期回档低点/高点
4. `2B` 作为更早、但不绝对的反转警告

这四层里：

1. `1-2-3` 是结构确认法
2. `2B` 是失败性新高/新低后的早期反转信号
3. 它们都不是“百分之百预测器”

### 3.5 寿命统计必须是连续分布，不是尾部老年刀

图 `11-1` 当前被正式定义为：

`中级主要走势的幅度与期限连续分布图`

它明确传达三件事：

1. 样本不是只看最老的尾部
2. 统计读法不是 `p65 / p95` 这种单边极端刀
3. 应按连续分布去看当前所处阶段

因此当前 canonical banding 改为：

1. `FIRST_QUARTER`
2. `SECOND_QUARTER`
3. `THIRD_QUARTER`
4. `FOURTH_QUARTER`

### 3.6 寿命不是纯时间年龄，而是时间 + 幅度

从 `160` 与 `345` 两张图共同收口后，当前冻结写定：

1. 寿命统计至少要有：
   - `duration`
   - `magnitude / extent`
2. 这两个维度必须形成：
   - 连续分布
   - 四分位读数
   - 联合寿命位置

### 3.7 平均寿命框架不是精准择时器，而是风险老化框架

图 `26-1` 与表 `26-1 / 26-2` 当前被写定为：

`用幅度与时间的相对位置，去估计当前走势已经老化到什么程度`

它的角色不是：

1. 精准预测某一根 bar 就见顶
2. 直接替代结构确认法

它的角色是：

1. 判断走势是否已经进入高龄区
2. 提高或降低你对后续延续的主观赔率
3. 与结构确认、市场背景、成交量关系一起构成风险框架

### 3.8 平均寿命 / odds 的 canonical 算式

当前冻结采用书里的中性表达，不直接写成“做多胜率”或“做空胜率”，而是先固定为：

1. `magnitude_remaining_prob`
   - 当前幅度之后，历史上还有多少同类走势继续走得更远
2. `duration_remaining_prob`
   - 当前时间之后，历史上还有多少同类走势继续走得更久
3. `lifespan_average_remaining_prob`
   - `magnitude_remaining_prob` 与 `duration_remaining_prob` 的均值
4. `lifespan_average_aged_prob`
   - `1 - lifespan_average_remaining_prob`
5. `lifespan_remaining_vs_aged_odds`
   - `remaining / aged`
6. `lifespan_aged_vs_remaining_odds`
   - `aged / remaining`

这样做的原因是：

1. 对 `MAINSTREAM` 波段，`aged` 更接近“主升/主跌段告一段落”的风险
2. 对 `COUNTERTREND` 波段，`aged` 更接近“修正段接近尾声、主趋势恢复”的概率
3. 同一套中性合同，可以避免在代码层过早硬编码交易方向解释

## 4. 对 Gene 的硬约束

从本文件起，Gene 后续重构必须同时满足：

1. 长期趋势与中级趋势分开定义
2. 中级主要走势与中级次级折返走势分开定义
3. 趋势改变必须同时支持：
   - `trendline break`
   - `1-2-3`
   - `2B`
4. canonical lifespan surface 不能继续用 `p65 / p95` 当主轴
5. canonical lifespan surface 必须支持：
   - `quartiles`
   - `duration`
   - `magnitude`
   - `joint lifespan`
6. 所有设计、代码、回测与记录，都必须明确自己消费的是：
   - `LONG`
   - `INTERMEDIATE`
   - `SHORT`
   - `MAINSTREAM`
   - `COUNTERTREND`

---

## 5. 一句话冻结

Gene 接下来要复刻的，不是“一个带 percentiles 的年龄因子”，而是：

`一套以长期趋势为背景、以中级主要走势为统计对象、以 1-2-3 / 2B 为趋势改变确认法、以 幅度 + 时间 四分位连续分布为寿命框架 的历史尺系统。`
