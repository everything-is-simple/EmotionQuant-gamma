## 1. 这份附录回答什么

这份附录只回答一个问题：

`如何把《专业投机原理》整本书中关于趋势对象、历史统计、转折确认、辅助确认与短期延续统计的思想，转译成 EmotionQuant 的第四战场。`

第四战场的职责不是直接回答“买不买”，而是回答：

`当前这只股票当前这段走势，在它自己的历史里，到底算什么级别。`

它是一把尺，不是一把枪。

## 2. 第四战场在系统中的位置

EmotionQuant 现在已经不是按一张总图纸顺滑长出来的系统，而是被真实问题逼出来的四个战场：

1. 第一战场：`PAS + MSS + IRS` 的初代系统框架，解决“先把框架搭起来”。
2. 第二战场：`Normandy / alpha`，解决“为什么打，打什么，何时打”。
3. 第三战场：`positioning / partial-exit / sizing`，解决“打多大，怎么退”。
4. 第四战场：`historical wave ruler`，解决“这段走势在历史里算什么级别”。

第四战场不直接替代第二、第三战场，但它会反过来给第二、第三战场补一层历史约束：

1. `BOF / PB / CPB` 不是一律等权出手，而要结合当前波段的历史位置。
2. `MSS / IRS` 未来要么重构为历史趋势尺，要么退役。
3. 第四战场先做“对象层”和“统计层”，后面才允许进入“条件层”。

## 3. 《专业投机原理》里真正该吸收什么

本书对第四战场真正有用的，不是某个震荡指标，也不是某个固定参数。前半本给出对象定义与原则框架，后半本补上历史分布、辅助确认和短期延续统计。

### 3.1 三种趋势同时存在

书中强调，同一市场同时存在：

1. 长期趋势
2. 中期趋势
3. 短期趋势

这意味着第四战场不能只给一条“单线判断”，而必须承认：

1. 同一只股票可以处于长期上升、中期回调、短期反抽的叠加状态。
2. `趋势` 不是一句话，而是一个带层级的对象。
3. 第四战场的第一版虽然可以先从日线中期波段切入，但术语必须一开始就支持多层级。

对应页段：

1. OCR 页 `50-51`，书内第 5 章开头，讨论三种趋势并列存在。

### 3.2 趋势先定义成结构，不先定义成指标

书中对趋势的核心定义非常朴素：

1. 上升趋势：高点与低点依次抬高。
2. 下降趋势：低点与高点依次降低。

这对第四战场的意义很大。它意味着：

1. 趋势对象的第一性定义必须是结构。
2. 指标、评分、z-score、百分位，都只能建立在结构已经被定义之后。
3. 第四战场不应该从“先算一个因子”开始，而应该从“先切出波段对象”开始。

对应页段：

1. OCR 页 `52-54`，书内第 5 章对上升趋势、下降趋势与验证的定义。

### 3.3 次级折返走势是“重要的逆流”

《专业投机原理》沿着道氏理论强调：

1. 趋势内部会有逆向折返。
2. 这些折返不是噪音，而是“重要的中期逆流”。
3. 重要折返有两个基本维度：`幅度` 与 `时间`。
4. 书中甚至给出经验区间：幅度多集中在前一主要波段的 `1/3 - 2/3`，时间多在 `3 周到 3 个月`。

对第四战场而言，这几乎就是原型：

1. 你提出“只采用波动幅度、波动时间”，方向是对的。
2. `趋势主流` 与 `趋势逆流` 的区分，正可以从这里抽象出来。
3. 书中提到的 `61% / 65% / 98%` 这类分布意识，说明它本来就不是只盯两个尾巴，而是用历史分布去理解“通常”。

对应页段：

1. OCR 页 `48-49`，书内第 4 章关于次级折返走势的幅度、时间与“重要性”。

### 3.4 1-2-3 与 2B 是转折确认法，不是神奇预测法

第 7 章最值钱的，不是图形名字，而是两种确认思维：

1. `1-2-3`：趋势线被破坏，不再创新高/新低，反向突破前一关键回档点。
2. `2B`：创新高/新低失败，随后跌回/涨回先前关键价位之内。

对第四战场而言：

1. `1-2-3` 应作为 `转折确认标签`。
2. `2B` 应作为 `失败性新高/新低事件标签`。
3. 它们都不应该在第一版被实现成复杂交易系统，而应该先成为历史数据库里的事件对象。

对应页段：

1. OCR 页 `73-75`，书内第 7 章 `1-2-3`。
2. OCR 页 `78-80`，书内第 7 章 `2B`。

### 3.5 震荡指标与宽度指标只做辅助，不做主宰

第 8 章的态度很重要：

1. 辅助指标可以帮助确认趋势转折。
2. 但它们只是辅助，不应喧宾夺主。
3. 作者明确强调 `KISS`，保持简单。

这正好给第四战场定边界：

1. 个股历史波段标尺的核心仍是 `波段幅度`、`持续时间`、`新高/新低事件结构`。
2. 指数、行业镜像尺后续可以借用 breadth 或 oscillators，但不应重回参数森林。

对应页段：

1. OCR 页 `98-101`，书内第 8 章关于宽度震荡指标、价格震荡指标和 `KISS`。

### 3.6 按目录重排后，后半本不是“附录”，而是统计层

这次按目录重看后，更准确的理解应当是：

1. 第 1 到 9 章给出第四战场的对象语言，重点是原则思考、道氏理论、趋势定义、`1-2-3` 与 `2B`。
2. 第 10 到 21 章给出风险、经济与经营哲学，它们不是第四战场的核心特征，但决定第四战场不能脱离风险视角单独运行。
3. 当前这个版本在 OCR 页 `308` 明确写着第 22、23 章为“略”，所以当前材料本身就是删节版，不能假装目录完整无缺。
4. 第 24 到 28 章其实是后半本最该进入系统的干货层，里面不只是“继续讲技术分析”，而是把波动、时间、平均寿命、宽度、相对强弱、成交量关系和几种确认法串成了概率框架。

按这次重读后的结果，第四战场不应再被理解成“前 100 页趋势定义的数据库化”，而应按下面次序吸收全书。

1. `G0`：先做价格对象层，只保留 `wave + event + ruler`，核心因子仍是 `magnitude / duration / extreme_density`。
2. `G1-G2`：把第 24、26 章的历史寿命与分布思想接进来，形成 `65% / 95%` 这类分布带，而不是尾部参数。
3. `G3`：把第 7、27 章的 `1-2-3 / 2B` 接成结构标签，而不是直接接成交易信号。
4. `G5`：把第 8、27、28 章的宽度、成交量、相对强弱、四天准则、缺口准则，吸收到指数/行业镜像尺。

## 4. 第四战场的核心使命

第四战场的正式使命建议定义为：

`为每一只股票建立一套可回看、可排序、可比较的历史波段标尺，用来判断当前波段相对于该股自身历史、相对于全市场同类波段、相对于所属行业与指数背景，处于什么级别。`

它先回答三个问题：

1. 当前波段在这只股票自己的历史里排第几。
2. 当前波段在全市场同类波段里排第几。
3. 幅度、时间、新高密度这三个子因子，谁对后续延续或衰竭更有解释力。

它暂时不直接回答：

1. 明天该不该买。
2. 明天该不该卖。
3. 仓位应该多大。

这些问题分别留给第二、第三战场。

## 5. 第四战场的术语冻结

### 5.0 术语分层

按当前治理冻结稿，第四战场正式采用四层分法：

1. 第一层，本体定义：
   `trend_level / trend / pivot / wave / reference extreme / trendline`
2. 第二层，转折确认语法：
   `1-2-3 / 2B`
3. 第三层，交易触发语法：
   `BOF / BPB / PB / TST / CPB`
4. 第四层，历史解释语法：
   `lifespan / percentile / band / mirror / conditioning`

后面的设计、实现、统计与交易触发，都只能建立在这条分层之上。

为了避免后面又回到一堆模糊词，第四战场建议先冻结下面这些术语。

### 5.1 趋势

`趋势` 是一个方向性的结构对象，而不是一个单值指标。

定义：

1. 上升趋势：确认高点抬高，确认低点抬高。
2. 下降趋势：确认高点降低，确认低点降低。
3. 盘整/等待行情：未形成可确认的高低点推进结构。

备注：

1. 趋势必须带 `level`，至少预留 `short / medium / long`。
2. 第四战场 G0 可以先只做 `medium`。

### 5.2 波段

`波段` 是两个相邻确认转折之间的单向价格段。

定义：

1. 每个波段都有起点、终点、方向、幅度、时间。
2. 波段必须来自交替出现的确认极值。
3. 波段是第四战场最基本的统计对象。

### 5.3 波段主流

`波段主流` 是顺应上一级趋势方向，并刷新上一级趋势极值的波段。

上升趋势中的主流波段：

1. 方向向上。
2. 刷新上一确认高点。
3. 不破坏上升趋势的结构定义。

下降趋势中的主流波段对称处理。

### 5.4 趋势逆流

`趋势逆流` 是逆于上一级趋势的波段，但尚未完成转折确认。

它对应《专业投机原理》里“次级折返走势”的现代化转译。

逆流只看两件事：

1. 幅度有多大。
2. 时间有多长。

### 5.5 转折

`转折` 不是一个感觉词，而是结构被确认改变的时刻。

第四战场建议区分两层：

1. `candidate_turn`：候选转折，结构开始松动。
2. `confirmed_turn`：确认转折，满足 `1-2-3` 或等价的结构确认。

### 5.6 新高 / 新低

`新高` 指价格刷新前一确认高点。  
`新低` 指价格刷新前一确认低点。

在第四战场里，新高/新低不只是一个布尔事件，而要记录：

1. 这是本波段内第几次新高/新低。
2. 发生时间。
3. 发生价格。
4. 累积数量。
5. 发生密度。

### 5.7 2B

补充冻结：

1. `2B` 的正式系统身份是第二层“转折确认语法”
2. 它是失败性极值刷新事件
3. 它不是第一层趋势本体
4. 它也不是第三层交易触发器

`2B` 是失败性新高/新低。

定义建议：

1. 上涨中，价格先创新高；
2. 但新高无法延续；
3. 在限定窗口内跌回先前关键高点之下；
4. 则记为 `2B_top`。

下跌中对称处理，记为 `2B_bottom`。

### 5.8 1-2-3

补充冻结：

1. `1-2-3` 的正式系统身份也是第二层“转折确认语法”
2. 它是趋势改变的三条件确认法
3. 它不是第一层趋势本体定义
4. 它也不是第三层交易触发器

`1-2-3 / 2B` 之所以必须保留，不是因为它们是趋势本体，
而是因为量化系统需要一套可回放、可审计的结构确认语法。

## 7. 第四战场只用哪三个核心子因子

第一版只建议保留三个核心子因子，不再扩张。

### 7.1 幅度因子 `magnitude`

定义：

1. 波段绝对涨跌幅
2. 波段对前一同向波段的扩张率
3. 波段对近 N 段自身分布的位置

意义：

1. 它回答“这一波到底走了多大”。

### 7.2 时间因子 `duration`

定义：

1. 波段持续的交易日数量
2. 当前波段在历史同向波段中的时间分位

意义：

1. 它回答“这一波到底走了多久”。

### 7.3 极值密度因子 `extreme_density`

它是你说的“涨波段内新高第几次、新高时间、新高价格、新高总数量、新高密度”的抽象名。

建议定义：

1. `extreme_count`：本波段内新高/新低累计次数
2. `extreme_density`：`extreme_count / duration_trade_days`
3. `extreme_first_gap`：波段起点到第一次新高/新低的间隔
4. `extreme_last_gap`：最后一次新高/新低距波段结束的间隔

意义：

1. 它回答“这一波是不是边走边刷新，而不是死撑着拖时间”。

## 8. 65% / 95% 的位置应该怎么用

你明确提出不想继续沿用只盯两个尾巴的 `IRS / MSS` 设计，这一点是对的。

第四战场里，`65% / 95%` 不应该被做成新参数森林，而应该被做成历史分布带。

建议定义：

1. `< P65`：普通波段
2. `P65 - P95`：强波段
3. `> P95`：极端波段

这三层恰好解决一个旧问题：

1. 不是只盯极端尾部；
2. 也不是把普通波段和强波段混在一起；
3. 可以把“强”和“极强”分开处理。

因此，第四战场里的 `65% / 95%` 不是新信号，而是：

`历史分布中的位置标签`

## 9. 第四战场如何吸收《专业投机原理》的 1-2-3 与 2B

### 9.1 1-2-3 的系统角色

`1-2-3` 不用于直接下单，而用于给波段末端打“确认转折”标签。

第四战场建议这样吸收：

3. `3`：反向突破前一关键回撤点

当 `1+2+3` 同时成立时：

1. 当前旧波段结束
2. 新波段确认开启
3. 旧趋势状态切换

### 9.2 2B 的系统角色

`2B` 不只是一个价格形态名，它是“失败性极值刷新”。

第四战场应该这样吸收：

1. 它属于 `event`
2. 它会影响 `turn_confirm_type`
3. 它在后续会成为第二战场信号的条件变量

换句话说：

1. 第二战场负责决定 `BOF / PB / CPB` 是否可打；
2. 第四战场负责提供 `当前这次 BOF/PB/CPB 所依附的波段级别与失败风险背景`。

这是第四战场必须保留的治理态度：

1. 能转译成历史趋势尺，就保留。

## 13. 结论

如果把《专业投机原理》真正读进第四战场，那么它给我们的不是“一个新指标”，而是下面这套秩序：

1. 趋势必须先被定义成结构。
2. 波段必须先被定义成对象。
3. 转折必须先被定义成确认，而不是猜测。
4. 新高/新低必须被当成事件序列，而不是单点标签。
5. 幅度和时间必须回到历史分布里理解。
6. 统计必须服务于结构，而不是反过来压扁结构。

所以，第四战场的本质，不是再造一个 `IRS / MSS` 式评分器，而是：

`为每只股票建立一把历史波段尺。`

有了这把尺，第二战场的信号、第三战场的出手与退出，才第一次有机会被放进“这段走势在历史里到底算什么”的真实语境里。
