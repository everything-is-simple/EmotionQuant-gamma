---

## 1. 目标

把《专业投机原理》里已经被判定为 `可进系统` 的材料，翻译成第四战场后续研发 backlog。

这张卡不直接开做全部实现，只完成四件事：

1. 明确 `G1 / G2 / G3 / G6` 各自回答的问题
2. 固定每一阶段的输入、输出与完成标准
3. 固定它们之间的依赖顺序
4. 防止后续把辅助层和人格层误塞进核心对象层

---

## 2. 翻译原则

本卡只消费以下来源：

1. `03-professional-speculation-principles-system-ingestion-matrix-20260316.md`
2. `02-professional-speculation-principles-map-20260316.md`
3. `02-professional-speculation-principles-theory-annex-20260316.md`

本卡固定遵守以下边界：

1. 只翻译 `可进系统`
2. 不把 `只做辅助` 提前上升为主定义
3. 不把 `只做人格约束` 误写成数据库字段或交易因子
4. 不在本卡中重开 `G4 / G5`，它们留待自历史验证和镜像尺阶段

## 3. 总体顺序

第四战场当前 backlog 顺序固定为：

1. `G1`: 三子因子解释力基线
2. `G2`: 历史寿命分布与 65/95 校准
3. `G3`: `1-2-3 / 2B` 结构标签校准
4. `G6`: `BOF / PB / CPB` 条件层统计

顺序不能颠倒。原因如下：

1. 没有 `G1`，我们不知道 `magnitude / duration / extreme_density` 谁最硬
2. 没有 `G2`，第四战场还没有真正的“历史尺”
3. 没有 `G3`，转折仍停留在粗糙摆点口径
4. 没有 `G6`，第二战场信号仍缺少历史波段条件约束

## 4. G1: 三子因子解释力基线

### 4.1 问题

回答：

`magnitude / duration / extreme_density` 这三个子因子，到底谁对后续延续或衰竭解释力更强。`

### 4.2 直接来源

1. 第4章关于 `幅度 + 时间` 的折返重要性，OCR `48-49`
2. 第11章关于风险可定量衡量，OCR `156-160`
3. 第26章关于走势年龄与胜算分布，OCR `344-350`

### 4.3 输入

1. `l3_gene_wave`
2. `l3_stock_gene`
3. 每个 completed wave 的 forward window 观察值

### 4.4 输出

1. `factor_eval` 基线表或等价研究产物
2. 三子因子分箱后的：
   - continuation rate
   - reversal rate
   - median forward return
   - median forward drawdown
   - monotonicity score
3. `G1` 实证卡结论：
   - 谁最强
   - 谁次强
   - 谁只能做补充

### 4.5 完成标准

1. 至少完成 `self-history` 口径下的三子因子排序
2. 至少完成一个固定 forward horizon 的解释力比较
3. 至少回答 `gene_score` 是否继续采用三项等权均值
4. 结论可复跑，不依赖人工挑样本

### 4.6 明确不做

1. 不引入新因子森林
2. 不在本卡重写波段定义
3. 不在本卡引入指数与行业镜像尺

## 5. G2: 历史寿命分布与 65/95 校准

### 5.1 问题

回答：

`当前这段波段，在它自己的历史分布里，是普通、强，还是极端。`

### 5.2 直接来源

1. 第24章关于交易机会与波动环境，OCR `310-318`
2. 第26章关于 `market average lifespan`、持续/终止概率，OCR `344-350`
3. 第29章关于窗口波动分布与中位数，OCR `404-418`

### 5.3 输入

1. `l3_gene_wave` 历史账本
2. 按方向拆分的 `amplitude_pct`
3. 按方向拆分的 `duration_trade_days`
4. 可选的窗口波动历史分布

### 5.4 输出

1. `lifespan distribution` 研究表
2. `P65 / P95` 校准口径
3. 新的 `wave_age_band` 或等价字段：
   - `<P65`
   - `P65-P95`
   - `>P95`
4. 当前波段持续/终止基础赔率的研究读数

### 5.5 完成标准

1. 幅度与时间都能输出自历史分布带
2. 能明确区分“普通波段”“强波段”“极端波段”
3. 至少给出一个可回写 `l3_stock_gene` 的年龄/分布标签
4. 不再只盯两个尾巴，而能保留中段强弱差异

### 5.6 明确不做

1. 不把 `65 / 95` 变成新的参数森林
2. 不把分布带直接等价于交易信号
3. 不在本卡处理 `1-2-3 / 2B` 精确定义

## 7. G6: BOF / PB / CPB 条件层统计

### 7.1 问题

回答：

`在接入第四战场历史尺之后，BOF / PB / CPB 在什么波段环境下更值得出手。`

### 7.2 直接来源

1. 第29章时间窗口分布，OCR `404-418`
2. 第30章连续上涨/下跌天数与后续收益，OCR `428-432`
3. 第28章 `三天高低价准则` 与战术确认，OCR `374-390`

### 7.3 输入

1. Normandy 主信号样本
2. `l3_stock_gene` 当前波段快照
3. `wave_age_band`
4. 连续上涨/下跌天数统计
5. `2B / 1-2-3 / gap` 等事件标签

### 7.4 输出

1. `BOF / PB / CPB` 条件层解释表
2. 各信号在不同历史尺位置下的：
   - hit rate
   - payoff ratio
   - median MAE/MFE
   - continuation / failure profile
3. 一份明确读数：
   - 早期第一次回调何时最好
   - 连续推进后何时衰竭风险抬升
   - `CPB` 在哪些环境下最容易误判

### 7.5 完成标准

1. 至少完成 `BOF / PB / CPB` 三类信号的条件分层比较
2. 至少有一项第四战场标签能显著改善其中一类信号读数
3. 结果可以回灌第二战场，但不强行改写其主信号定义

### 7.6 明确不做

1. 不在本卡重做 Normandy alpha provenance
2. 不在本卡讨论 partial-exit
3. 不把所有条件层读数直接塞成硬过滤器

## 8. 阶段依赖

阶段依赖固定如下：

1. `G1 -> G2`
   - 先确认三子因子，再做历史尺分布带
2. `G2 -> G3`
   - 先有年龄/分布框架，再判断转折标签在什么年龄段更可信
3. `G3 -> G6`
   - 先把事件标签做稳，再回灌到 `BOF / PB / CPB`

任何试图跳过依赖的做法，都容易把第四战场重新做成“描述性文档”，而不是生产工具。

## 10. 当前结论

从《专业投机原理》翻译进第四战场之后，最值得马上开做的不是更多定义，而是下面这条流水线：

1. 先证明三子因子谁最硬
2. 再把波段做成年龄和分布带
3. 再把 `1-2-3 / 2B` 从描述词做成正式标签
4. 最后再回去约束 `BOF / PB / CPB`

这就是第四战场从“读书启发”走向“系统 backlog”的最短路径。
