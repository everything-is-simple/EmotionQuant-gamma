## 2. 当前已定位的核心章节

### 2.1 第4章：道氏理论与修正走势

当前定位页：`47-49`

这一段对第四战场最有价值的不是“指数预测”，而是三件事：

1. 趋势天然分层：长期 / 中期 / 短期
2. 修正走势要同时看幅度与持续时间
3. “重要修正”与“趋势结束”之间，本来就存在主观而困难的边界

这直接对应第四战场里：

1. `趋势`
2. `波段主流`
3. `趋势逆流`
4. 幅度与时间的双维统计

另外，第4章明确提到对历史趋势做归类、衡量与列表统计，这与第四战场要建的历史波段数据库高度同构。

### 2.2 第5章：趋势的定义

当前定位页：`50-58`

这一章是第四战场的第一术语来源。这里可以直接抽出：

1. 上升趋势：高点与低点持续抬高
2. 下降趋势：低点与高点持续下移
3. 趋势判断必须先明确时间层级
4. 盘整、突破、确认，都要有一致性的口径

第四战场第一版应优先吸收这里的对象定义，而不要先引入过多指标。

对应关系如下：

1. `趋势` -> 高低点结构
2. `新高 / 新低` -> 趋势极值刷新事件
3. `波段` -> 相邻反向 pivot 之间的单向段
4. `转折前状态` -> 盘整、失败突破、确认突破

### 2.3 第7章：辨识趋势改变，1-2-3 与 2B

当前定位页：`73-80`

这是第四战场第二术语来源，也是转折定义的主来源。

从这一段里，第四战场要抽出的不是整套交易法，而是两种事件定义：

1. `1-2-3`
   - 趋势线或推进结构被破坏
   - 不再延续原方向创新高 / 创新低
   - 反向突破最近一次重要回撤点
2. `2B`
   - 创新高 / 新低失败
   - 失败后回落 / 回升并跌回先前关键位内

这两者在第四战场中的位置应该是：

1. `1-2-3` 用来定义 `confirmed reversal`
2. `2B` 用来定义 `failed extreme event`

它们首先是数据库事件标签，其次才可能成为交易条件。

### 2.4 第8章：宽度、震荡与统计辅助

当前定位页：`98-105`

这一段不应直接照搬进个股价格标尺，但它有两点非常关键：

1. 指数与 breadth 的确认，可以作为后续 `market / industry / index mirror ruler` 的素材
2. 作者把统计工具放在趋势对象之后，而不是放在趋势定义之前

这很重要，因为它支持第四战场当前的路线：

1. 先定义价格对象
2. 再做 percentile / z-score
3. 最后才决定是否引入指数级镜像尺

## 3. 第四战场当前应抽取什么，不应抽取什么

### 3.1 当前应抽取

1. 趋势与修正的结构定义
2. `1-2-3` 的确认逻辑
3. `2B` 的失败性新高 / 新低逻辑
4. 幅度与时间作为最低限度的统计原语
5. “先有对象，再有统计”的顺序

### 3.2 当前不应直接抽取

1. 整套指数预测框架
2. 过多的震荡指标参数
3. 把个股历史标尺直接做成交易信号机
4. 在第一版就把 `MSS / IRS` 全部改写进去

当前第四战场的正确边界仍然是：

`先做个股 price-only historical wave ruler，再决定 MSS / IRS 是改造还是退役。`

---

## 4. 对第四战场最直接的落地启发

根据目前定位，第四战场第一版术语冻结应优先围绕六个对象展开：

1. `趋势`
2. `波段`
3. `波段主流`
4. `趋势逆流`
5. `转折`
6. `新高 / 新低`

第一版统计只保留三类子因子：

1. `magnitude`
2. `duration`
3. `extreme_density`

然后把 `1-2-3` 与 `2B` 作为事件标签，贴到历史波段和极值事件上，而不是先把它们写成复杂信号。

---

## 5. 当前结论

《专业投机原理》对第四战场有用，但用途必须被限定。

它最有用的部分，不是拿来替代我们的系统，而是帮助我们把下列对象定义得更扎实：

1. 趋势
2. 修正
3. 转折
4. 失败性极值
5. 指数级确认与统计镜像

所以，这本书对第四战场的价值，主要是：

`提供对象学骨架，而不是提供现成系统。`
