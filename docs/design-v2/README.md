# design-v2 目录说明

## 定位

`docs/design-v2/` 只存放系统级设计基线与模块级设计，不存放分阶段执行材料。

## 结构

1. 系统基线（SoT）
   - `system-baseline.md`
2. 模块级设计
   - `architecture-master.md`
   - `data-layer-design.md`
   - `selector-design.md`
   - `strategy-design.md`
   - `broker-design.md`
   - `backtest-report-design.md`
3. 评审标准
   - `sandbox-review-standard.md`
4. 阶段材料归属
   - 分阶段文件统一归档到 `docs/spec/<version>/`
