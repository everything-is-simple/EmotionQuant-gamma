# docs/navigation 导航层

## 定位

`docs/navigation/` 用来提供额外的人类阅读入口。

这里不承载正式设计正文，也不承载证据归档；它只负责把已经存在的文档，按更容易理解的方式重新组织出一层阅读路径。

## 当前入口

- `four-battlefields-document-shelf/README.md`

## 权威入口

- `../design-v2/01-system/system-baseline.md`
- `../spec/common/records/development-status.md`

## 维护规则

1. 这里优先做非破坏式重组，不直接移动旧文档。
2. 如果旧文档已有稳定路径，导航层只增加索引，不重写原文件职责。
3. 导航层的说明必须服从 `blueprint/`、`docs/spec/` 和各战场 README 的正式口径。
