## 定位

`docs/reference/code-maps/` 保存“现行代码地图”。

这里回答的是：
1. 某一层代码现在有哪些文件
2. 每个文件读哪些表、写哪些表
3. 模块之间怎样串成一条实际走线

这里不回答：
1. 历史 frozen 设计为什么这样
2. 当前主线治理状态是什么
3. 某张执行卡是否完成
