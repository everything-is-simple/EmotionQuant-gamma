## 📋 分类说明

### 1. `a-stock-rules/` - A股市场规则

存放 A 股市场相关的交易规则、制度说明和行业分类标准：

- 交易时间、涨跌幅限制、停牌规则
- 涨跌停板制度（主板/科创板/创业板）
- 申万行业分类标准（一级/二级/三级）

**使用场景**：
- 开发交易逻辑时查阅规则
- 测试验证代码符合 A 股规则
- IRS 模块行业轮动分析

### 2. `operations/` - 运维操作指南

存放系统运维、开发环境管理相关的操作指南：

- 临时文件管理（缓存、配置、产物）
- 清理策略与最佳实践
- 环境检查与问题排查

**使用场景**：
- 定期清理临时文件
- 提交前检查
- 问题排查
