## 🎯 使用场景

1. **开发参考**：编写交易逻辑时查阅规则
2. **测试验证**：确保代码符合 A 股规则
3. **行业分析**：IRS 模块行业轮动分析

## ⚠️ 注意事项

1. **时效性**：规则可能随监管政策调整，需定期更新
2. **数据源**：基于 TuShare 接口，与实际交易所规则保持一致
3. **优先级**：如有冲突，以交易所官方规则为准
