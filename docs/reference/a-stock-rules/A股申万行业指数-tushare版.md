## 申万行业分类参考

申万行业分类是 A 股常用行业分类标准之一。仓库当前参考口径是：

1. `SW2021`
2. 一级行业用于当前主线的主要行业映射参考
3. 更细级别分类可作为数据补充，但是否消费取决于当前设计版本

### 多级结构

| 层级 | 常见标识 | 用途 |
|---|---|---|
| `L1` | 一级行业 | 当前主线最常见的行业聚合口径 |
| `L2` | 二级行业 | 更细分的行业研究与扩展参考 |
| `L3` | 三级行业 | 更细颗粒度研究参考 |

### 一级行业示意

当前常见一级行业包括但不限于：

- 农林牧渔
- 基础化工
- 钢铁
- 有色金属
- 电子
- 汽车
- 家用电器
- 食品饮料
- 医药生物
- 计算机
- 通信
- 传媒
- 电力设备
- 国防军工
- 银行
- 非银金融

说明：

1. 具体行业数量与名称以官方当期口径为准。
2. 仓库内若做行业名称标准化，应保留“外部原值”和“内部使用值”的映射关系。

## TuShare 常见接口参考

### 1. 行业分类层级

常见接口示意：

```python
import tushare as ts

pro = ts.pro_api("YOUR_TOKEN")

df_l1 = pro.index_classify(level="L1", src="SW2021")
df_l2 = pro.index_classify(level="L2", src="SW2021")
df_l3 = pro.index_classify(level="L3", src="SW2021")
```

用途：

1. 获取申万行业层级字典
2. 建立 `industry_code <-> industry_name` 映射
3. 为本地静态维表更新提供参考

### 2. 行业指数日线

常见接口示意：

```python
df_sw_daily = pro.sw_daily(trade_date="20250307")
```

用途：

1. 获取行业指数日线
2. 做外部校验或补充对照
3. 为行业趋势研究提供参考输入

### 3. 行业成分映射

仓库若需要行业成员映射，应关注：

1. 成分股代码格式
2. 生效日期 / 失效日期
3. 行业调整带来的历史回看一致性

说明：

1. 行业成员映射是“历史正确性”关键维表。
2. 若存在名称与代码冲突，优先保留原始来源值，再在下游做标准化。
