# 第一战役：v0.01 Frozen

## 战役定位

这里收 `v0.01` 这场历史战役的统一入口。

它已经冻结，不再承载当前主线，也不再承担研究线的新实验。

## 当前裁决

1. `v0.01` 已结束，当前定位固定为 `historical baseline + rollback reference + failure experience archive`。
2. 它不再承担当前主线指挥，也不再承接新的研究任务。
3. 任何对 `v0.01` 的回看，目的都只能是复盘历史边界、提炼失败经验和保留对照尺。

## 已淘汰路线

1. 把 `v0.01` 继续当成当前主线版本推进。
2. 直接复活 `v0.01` 的后续改进方案，绕开当前主线和研究线治理。
3. 让 `v0.01` 历史设计继续承担今天的系统裁决职责。

## 下一步

1. 只在需要历史对照、失败复盘和回退参考时进入本战役。
2. 当前设计问题转到 `blueprint/`，当前研究问题转到 `normandy/`。
