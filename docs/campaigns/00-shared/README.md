# 共用层

## 定位

这里收跨战役共用资产。

这里不是任何一场战役的主入口，也不存放某一场战役的专属实验卡、失败路线或临时计划。

## 当前入口

| 类型 | 路径 | 用途 |
|---|---|---|
| 跨线治理 | `../../spec/common/README.md` | 当前状态、系统宪法、仓库三线地图 |
| 参考资料 | `../../reference/README.md` | 外部规则、运维与长期参考 |
| 工作流 | `../../workflow/README.md` | 固定执行流程 |
| 观察框架 | `../../observatory/README.md` | 评审标准与高层观察框架 |
| 理论来源 | `../../Strategy/README.md` | MSS / IRS / PAS 的方法论来源 |

## 一句话边界

`属于所有战役都要共用的东西，才留在这里；只属于某一战役的文件，必须迁回该战役。`
