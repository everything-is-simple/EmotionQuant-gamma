# docs/spec 目录总索引

## 目录定位

`docs/spec/` 存放所有分阶段材料（v0.01+），包括：

1. 路线图
2. 实现卡
3. 阶段 runbook
4. 阶段勘误与发布记录
5. 阶段评审与回归证据

## 目录结构

1. `v0.01/` - v0.01 阶段全量材料
2. `v0.02/` - v0.02 阶段全量材料
3. `v0.03/` - v0.03 阶段全量材料
4. `v0.04/` - v0.04 阶段全量材料
5. `v0.05/` - v0.05 阶段全量材料
6. `v0.06/` - v0.06 阶段全量材料
7. `common/` - 跨阶段共用材料

## 配套目录

1. `docs/design-v2/`：系统级总设计与模块设计（不放阶段材料）
2. `docs/spec/common/records/`：跨阶段治理记录（development-status / debts / reusable-assets）
