# v0.05 路线图

**版本**: v0.05 草案  
**状态**: Draft（待 v0.04 完成后细化）  
**前置依赖**: v0.04 通过 Gate 验收  
**核心使命**: 策略生态可进化（注册表生态管理）

---

## 1. 核心交付

1. **注册表生态管理**：滚动期望值/重叠度控制/环境启用矩阵
2. **信号强度有效性验证**：strength 与实际表现的相关性分析
3. **形态自动降级/升级**：基于滚动期望值的形态启停机制
4. **环境适配矩阵**：不同形态在不同环境下的启用策略

---

## 2. 晋级门槛

v0.04 → v0.05 必须同时满足：

1. v0.04 连续两个评估窗通过且无强制回退
2. 注册表生态管理不破坏现有形态表现
3. 形态自动降级/升级机制可复现
