# v0.06 路线图

**版本**: v0.06 草案  
**状态**: Draft（待 v0.05 完成后细化）  
**前置依赖**: v0.05 通过 Gate 验收  
**核心使命**: 组合层闭环（总风险预算 + 暴露限制）

---

## 1. 核心交付

1. **总风险预算**：账户级风险上限 + 动态调整
2. **行业/主题暴露限制**：单行业/单主题仓位上限
3. **动态降仓**：市场环境恶化时自动降低仓位
4. **候选池分层**：ACTIVE/OBSERVE/BACKUP 三层候选池管理

---

## 2. 晋级门槛

v0.05 → v0.06 必须同时满足：

1. v0.05 连续两个评估窗通过且无强制回退
2. 组合层风控不破坏单形态表现
3. 动态降仓机制可复现
