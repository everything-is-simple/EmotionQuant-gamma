# v0.04 路线图

**版本**: v0.04 草案  
**状态**: Draft（待 v0.03 完成后细化）  
**前置依赖**: v0.03 通过 Gate 验收  
**核心使命**: 用统计替代感觉（监控覆盖层）

---

## 1. 核心交付

1. **更细的环境分桶矩阵**：不只 BULLISH/NEUTRAL/BEARISH，细分到波动率/成交额/涨跌分布
2. **执行摩擦统计**：滑点/冲击成本/流动性衰减实测
3. **风险归因**：分形态/分环境/分行业的风险贡献分解
4. **监控覆盖层**：叠加到现有 report，不改变交易逻辑

---

## 2. 晋级门槛

v0.03 → v0.04 必须同时满足：

1. v0.03 连续两个评估窗通过且无强制回退
2. 环境分桶矩阵可复现
3. 执行摩擦统计不改变回测结果（只增加观测维度）
