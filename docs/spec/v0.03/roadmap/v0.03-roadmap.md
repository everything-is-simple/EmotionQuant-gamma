# v0.03 路线图

**版本**: v0.03 草案  
**状态**: Draft（待 v0.02 完成后细化）  
**前置依赖**: v0.02 通过 Gate 验收  
**核心使命**: 完整 YTC 五形态 + 组合模式评估

---

## 1. 核心交付

1. **pas_tst.py**：TST 支撑测试（DD 双十字星 + SB 二次突破）
2. **pas_pb.py**：PB 趋势回调
3. **pas_cpb.py**：CPB 复杂回调（SB + ARB，M/W 形态 + 头肩底）
4. **PAS_COMBINATION 生效**：ANY/ALL/VOTE 三种组合模式评估
5. **CriticalPointManager**：临界点动态管理（Volman 第 14 章适配）
6. **IRS 龙头 + 基因库因子**：从 4 因子扩展到 6 因子
7. **不利条件预警**：Volman 第 15 章（阻力位/冲击效应/真空效应）

---

## 2. 晋级门槛

v0.02 → v0.03 必须同时满足：

1. v0.02 连续两个评估窗通过且无强制回退
2. TST/PB/CPB 各自在单形态回测中先独立通过
3. 五形态组合模式不破坏最佳单形态表现
