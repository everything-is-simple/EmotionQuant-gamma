# v0.02 路线图

**版本**: v0.02 草案  
**状态**: Draft（待 v0.01 完成后细化）  
**前置依赖**: v0.01 通过 Gate 验收  
**核心使命**: 多形态并行评估（BPB + BOF）

---

## 1. 核心交付

1. **pas_bpb.py**：BPB 突破回踩形态检测器
2. **gene.py 实现**：基于 v0.01 历史样本做事后反推，形成可执行定义
3. **IRS 扩展因子**：从 2 因子扩展到 4 因子（连续性 + 估值）
4. **BB 箱体增强**：所有形态的信号强度加权

---

## 2. 晋级门槛

v0.01 → v0.02 必须同时满足：

1. v0.01 连续两个评估窗通过 §6.2 且无强制回退
2. BPB 在单形态回测中先独立通过（EV ≥ 0, PF ≥ 1.05）
3. BPB + BOF 组合模式不破坏 BOF baseline 表现
