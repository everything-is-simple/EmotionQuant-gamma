# Spec 文档入口（统一规则）

## 存放规则

1. `docs/spec/` 是全量归档目录：存放所有版本（v0.01+）阶段性材料。
2. 每个版本使用独立子目录：`docs/spec/<version>/`。
3. 跨版本文件放在 `docs/spec/common/`。
4. 每个阶段目录可包含：路线图、实现卡、勘误、runbook、发布记录、评审证据等。
