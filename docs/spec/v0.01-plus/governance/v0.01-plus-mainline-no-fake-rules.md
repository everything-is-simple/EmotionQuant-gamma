## 1. 目标

本文只做一件事：

`把主线当前不允许冒充的对象、口径和自动动作统一写死。`

---

## 2. 当前唯一允许的真实口径

1. 当前治理主线是 `v0.01-plus`。
2. 当前默认运行路径仍是 `legacy_bof_baseline`。
3. 第二、第三、第四战场可迁回主线的，当前只包括：
   - `governance boundary`
   - `execution scaffold`
   - `control hierarchy`
   - `negative constraints`
   - `context sidecar boundary`
4. 第二、第三、第四战场当前都没有产出可直接改写主线默认参数的 formal promotion。

## 4. 不允许自动发生的动作

1. 不自动把研究线 retained / watch 对象升格成主线默认参数。
2. 不自动打开 `PX1 / cross-exit sensitivity`。
3. 不自动打开 `PX2 / targeted mechanism follow-up`。
4. 不自动打开任何新的 `targeted mechanism hypothesis`。
5. 不自动把 `same-code cooldown signal_filter hook` 升格为默认开启。
6. 不自动把 `R2 / R3 / R8 / R9` 或任何仍受 `ALREADY_HOLDING` 阻塞的机制段带回主线。

## 5. 允许说的话与不允许说的话

### 5.1 允许说的话

1. `v0.01-plus` 是当前治理主线。
2. 当前默认运行路径仍是 `legacy_bof_baseline`。
3. `FIXED_NOTIONAL_CONTROL / SINGLE_LOT_CONTROL / FULL_EXIT_CONTROL` 是当前主线已经吸收的 control hierarchy。
4. `TRAIL_SCALE_OUT_25_75` 是当前 retained queue 里的 provisional leader。
5. `Tachibana pilot-pack` 是当前受治理约束的 executable boundary。
6. `Gene` 当前只允许以 `context sidecar / dashboard / attribution` 身份进入主线候选。

1. “`DTT` 已经是当前默认运行路径。”
2. “完整 Tachibana 系统已经 ready。”
3. “`TRAIL_SCALE_OUT_25_75` 已经是 canonical control。”
4. “`CD10` 已经是默认 cooldown。”
5. “`SINGLE_LOT_CONTROL` 已经是一条新的 operating lane。”
6. “`PX1 / PX2` 就是下一步自动要开的卡。”
7. “`Gene` 已经是默认 runtime filter / hard gate。”
