## 2. 四个必答问题

1. `BPB` 在本窗口没有形成成交样本。
2. `PB` 新增了 `54` 个选中 signal、`26` 笔买入；代表样本包括：
   - `002371`
   - `000100`
   - `300308`
   - `603799`
   - `600460`
3. `TST` 新增了 `29` 个选中 signal、`18` 笔买入；代表样本包括：
   - `002757`
   - `300894`
   - `000650`

4. `CPB` 是新增形态里的主贡献者，新带来 `193` 个选中 signal、`32` 笔买入；代表样本包括：
   - `002230`

5. `YTC5_ANY` 相对 `BOF` 总共新增了 `279` 个选中 signal、`35` 笔买入，新增部分几乎全部由 `CPB / PB / TST` 拉动。

### 2.2 哪些票是因为 `quality` 被筛下去或排上来

结论：

1. 当前没有任何票因为 `quality` 被筛下去或排上来。
2. `YTC5_ANY` 与 `YTC5_ANY + quality` 的：
   - `rank_diff_days = 0`
   - `execution_diff_days = 0`
   - 新增选中 signal = `0`
   - 新增买入 = `0`
3. 这与当前主线边界一致：`quality` 只进入 `PAS trace / sidecar`，尚未进入 formal `Signal` 或 `final_score`。
4. 但 `quality` 解释层本身已经完整落地：
   - `quality_trace_coverage = 1.0`
   - `reference_trace_coverage = 1.0`
5. 当前 top quality 样本已可直接从证据里追溯，例如：
   - `600340 / bof / 95.36`
   - `002227 / bof / 90.17`
   - `002759 / bof / 89.68`

### 2.3 哪个形态在什么环境桶下有效

结论：

1. `PB / TST / CPB / YTC5_ANY` 的有效样本在这轮窗口里都集中落在 `NEUTRAL`。
2. `BOF` 的环境拆分里同时出现了 `BULLISH` 和 `NEUTRAL`，但新增形态的有效样本没有形成可对比的多桶分布。
3. 因此，这轮证据只能给出一个保守判断：
   - `CPB` 在当前 `NEUTRAL` 桶里表现最好
   - `PB` 在当前 `NEUTRAL` 桶里次之
   - `TST` 在当前 `NEUTRAL` 桶里为负贡献
4. 这还不足以支持“按环境桶启停某个形态”的主线决策；后续需要在 `Phase 3 / MSS` 后再做更可信的环境切片。

### 2.4 单形态和组合形态的执行摩擦是否不同

结论：不同，而且差异已经很明显。

1. `BOF` 的执行摩擦最低：
   - `reject_rate = 0`
   - `participation_rate = 1.0`
   - 但收益表现最弱
2. `PB` 已出现明显摩擦：
   - `reject_rate = 0.3418`
   - `skip_maxpos_count = 22`
3. `TST` 的摩擦略低于 `PB`，但收益明显更差：
   - `reject_rate = 0.2174`
   - `participation_rate = 0.6207`
4. `CPB` 的收益最强，但摩擦也最大：
   - `reject_rate = 0.7156`
   - `skip_maxpos_count = 160`
   - `participation_rate = 0.1658`
5. 组合 `YTC5_ANY` 的摩擦比任何单形态都更重：
   - `reject_rate = 0.7764`
   - `skip_maxpos_count = 241`
   - `participation_rate = 0.1250`
6. 这说明新增形态已经明显改变交易集合，但“组合后一定更好”并不成立；`CPB` 的 alpha 与组合后的拥挤摩擦同时存在。

## 3. 当前裁决

### 3.1 可以确认的结论

1. `Phase 1 / PAS` 的五形态 registry、quality sidecar、reference layer 与专项 evidence 已全部落地。
2. `BOF` 基线没有被破坏，专项 evidence 里仍能单独重跑并稳定输出。
3. `quality` 当前已达到“解释层完整、运行时零侵入”的目标。
4. `CPB / PB` 已经证明自己会显著改变交易集合和收益结构。

### 3.2 暂时不写死的结论

1. 暂不把 `YTC5_ANY` 直接升为新的默认 PAS 运行形态集合。
2. 暂不让 `quality` 直接进入 `final_score`。
3. 暂不基于当前证据对形态做环境桶启停。

### 3.3 下一步

1. `Phase 1` 可以出场。
2. 主线继续进入 `Phase 2 / IRS`。
3. `PAS` 后续只保留两类待决项：
   - 是否让 `quality` 进入排序分
