## 3. 本轮回答的三个问题

### 3.1 IRS 五层是否真的改写了排序

结论：`是`。

1. `IRS_RSRV` 与 `IRS_RSRVRTBDGN` 相对 `IRS_LITE` 都在 `112 / 112` 个交易日上改写了行业排名。
2. 这说明 `RS / RV / RT / BD / GN` 已经进入正式排序链，而不是只写 trace 或 sidecar。

### 3.2 排序变化是否穿透到了执行层

结论：`是，但幅度有限`。

1. 长窗口里，`IRS_RSRV` 和 `IRS_RSRVRTBDGN` 相对 `IRS_LITE` 都出现了 `execution_diff_days_vs_lite = 1`。
2. 这比短窗口的 `0` 已经更强，说明排序变化不只是停留在行业 rank 层。
3. 但变化仍然很有限，说明当前 `BOF-only + 当前仓位容量 + 当前窗口信号密度` 仍然压缩了 IRS 变化对执行集合的穿透度。

### 3.3 IRS_RSRVRTBDGN 是否已经证明优于 IRS_LITE

结论：`还不能这样写`。

1. `IRS_RSRV / IRS_RSRVRTBDGN` 相对 `IRS_LITE` 的 `EV` 和 `MDD` 有改善。
2. 但 `profit_factor` 并未同步改善。
3. `IRS_RSRVRTBDGN` 相对 `IRS_RSRV` 的执行结果基本持平，说明五层全开在这段窗口里还没有给出足够强的增量优势证据。
