### 2.2 行业分组

根据行业特征，可以分为以下几大类：

#### 周期类（10个）
- 采掘、化工、钢铁、有色金属、建筑材料、建筑装饰、机械设备、汽车、石油石化、房地产

**特征**：
- 与宏观经济周期高度相关
- 盈利波动大
- 估值波动大
- 适合在经济复苏期配置

#### 消费类（7个）
- 农林牧渔、家用电器、食品饮料、纺织服饰、轻工制造、商贸零售、社会服务、美容护理

**特征**：
- 需求相对稳定
- 盈利稳定增长
- 防御性强
- 适合在经济下行期配置

#### 成长类（8个）
- 电子、医药生物、电力设备、国防军工、计算机、传媒、通信、环保

**特征**：
- 高成长性
- 高估值
- 高波动
- 适合在流动性宽松期配置

#### 金融类（2个）
- 银行、非银金融

**特征**：
- 高杠杆
- 高分红
- 低估值
- 适合在利率上行期配置

#### 公用事业类（2个）
- 公用事业、交通运输

**特征**：
- 稳定现金流
- 稳定分红
- 防御性强
- 适合在市场波动期配置

#### 综合类（1个）
- 综合

**特征**：
- 多元化经营
- 难以归类
- 通常不作为主要配置

## 3. 行业生命周期分类

### 3.1 生命周期理论

根据行业发展阶段，可以分为：

#### 成长期行业（Growth）
**特征**：
- 高增长、高估值
- 看重PB（资产重估）
- 对利率敏感

**代表行业**：
- 电子、计算机、通信、传媒
- 电力设备（新能源）
- 医药生物（创新药）
- 国防军工

**估值逻辑**：
- PE权重：0.35
- PB权重：0.65
- 原因：成长股更看重资产重估和未来预期

#### 均衡期行业（Balanced）
**特征**：
- 稳定增长、合理估值
- PE和PB并重

**代表行业**：
- 食品饮料、医药生物（成熟药企）
- 家用电器、轻工制造
- 社会服务、美容护理

**估值逻辑**：
- PE权重：0.50
- PB权重：0.50
- 原因：均衡成长，PE和PB都重要

#### 价值期行业（Value）
**特征**：
- 低增长、低估值
- 看重PE（盈利修复）
- 高分红

**代表行业**：
- 银行、非银金融
- 房地产、建筑材料、建筑装饰
- 钢铁、煤炭、石油石化
- 公用事业、交通运输

**估值逻辑**：
- PE权重：0.65
- PB权重：0.35
- 原因：价值股更看重盈利修复和分红

## 4. 个股行业归属规则

### 4.1 归属原则

**主营业务原则**：
- 根据公司主营业务收入占比确定行业归属
- 主营业务收入占比>50%的业务确定行业

**动态调整原则**：
- 每季度根据公司财报调整
- 主营业务发生重大变化时调整

**唯一归属原则**：
- 每只股票只能归属一个一级行业
- 多元化公司归入"综合"行业

### 4.2 特殊情况处理

**ST股票**：
- 保留原行业归属
- 但在行业统计时可选择排除

**新股**：
- 上市时确定行业归属
- 上市N日内（如20日）可选择排除

**停牌股票**：
- 保留行业归属
- 但在行业统计时自动排除

**退市股票**：
- 从行业成分股中移除

## 5. 行业指数体系

### 5.1 申万一级行业指数

**指数代码规则**：
- 格式：`801XXX.SI`
- 例如：`801010.SI`（农林牧渔指数）

**指数编制方法**：
- 成分股：该行业的所有A股
- 权重：自由流通市值加权
- 调整频率：每季度调整一次

### 5.2 行业指数的用途

**在IRS中的应用**：
1. **相对强度计算**：
   - 行业指数涨跌幅 vs 基准指数涨跌幅
   - 计算超额收益

2. **趋势判断**：
   - 行业指数的均线系统
   - 行业指数的新高新低

3. **估值分析**：
   - 行业指数的PE、PB
   - 行业估值的历史分位

## 6. 数据来源与更新

### 6.1 TuShare数据接口

**行业分类接口**：
```python
# 获取申万行业分类
df = pro.index_classify(
    level='L1',  # 一级行业
    src='SW2021'  # 申万2021版
)

# 获取个股行业归属
df = pro.stock_basic(
    fields='ts_code,name,industry'
)

# 获取行业成分股
df = pro.index_member(
    index_code='801010.SI'  # 农林牧渔指数
)
```

**行业指数数据**：
```python
# 获取行业指数日线数据
df = pro.index_daily(
    ts_code='801010.SI',
    start_date='20200101',
    end_date='20241231'
)
```

### 6.2 更新频率

**行业归属更新**：
- 频率：每季度
- 时间：财报披露后
- 来源：申万官方调整

**行业指数更新**：
- 频率：每日
- 时间：收盘后
- 来源：TuShare API

## 7. 与EmotionQuant的映射

### 7.1 IRS模块的使用

**行业范围**：
- 使用申万一级31个行业
- 不使用二级、三级分类（流动性不足）

**行业评分**：
- 基于行业指数的相对强度
- 基于行业成分股的广度指标
- 基于行业资金流向
- 基于行业估值水平

**行业轮动**：
- 识别强势行业（超配）
- 识别弱势行业（减配）
- 识别轮动趋势（IN/OUT/HOLD）

### 7.2 数据存储

**L1层（原始数据）**：
- `raw_index_classify`：行业分类表
- `raw_index_member`：行业成分股表
- `raw_index_daily`：行业指数日线

**L2层（加工数据）**：
- `industry_snapshot`：行业快照（每日）
- 包含：行业涨跌幅、成交额、PE/PB、成分股统计等

**L3层（算法输出）**：
- `irs_industry_score`：行业评分
- 包含：综合评分、轮动状态、配置建议等

## 8. 行业轮动的历史规律

### 8.1 牛市初期（萌芽期-发酵期）

**领涨行业**：
- 券商（非银金融）：牛市旗手
- 电子、计算机：科技成长
- 电力设备：新能源

**原因**：
- 流动性宽松
- 风险偏好提升
- 成长股估值修复

### 8.2 牛市中期（加速期-分歧期）

**领涨行业**：
- 食品饮料、医药生物：核心资产
- 家用电器、美容护理：消费升级
- 新能源、半导体：主题投资

**原因**：
- 盈利驱动
- 机构抱团
- 赚钱效应扩散

### 8.3 牛市末期（高潮期）

**领涨行业**：
- 周期股：钢铁、煤炭、有色
- 小盘股：题材炒作
- 垃圾股：炒壳

**原因**：
- 流动性泛滥
- 风险偏好极高
- 补涨行情

### 8.4 熊市初期（扩散期）

**抗跌行业**：
- 银行：高分红、低估值
- 公用事业：防御性
- 食品饮料：必需消费

**原因**：
- 流动性收紧
- 风险偏好下降
- 资金避险

### 8.5 熊市中期（退潮期）

**抗跌行业**：
- 银行、公用事业：防御性
- 黄金、农业：避险

**原因**：
- 经济下行
- 盈利下滑
- 估值压缩

## 9. 行业配置策略

### 9.1 基于市场情绪的配置

**MSS高温（>75°C）**：
- 减配周期股（钢铁、煤炭、有色）
- 减配题材股（传媒、计算机）
- 增配防御股（银行、公用事业）

**MSS中温（45-75°C）**：
- 标配核心资产（食品饮料、医药、家电）
- 标配成长股（电子、新能源）
- 减配周期股

**MSS低温（<30°C）**：
- 超配成长股（电子、计算机、新能源）
- 超配券商（非银金融）
- 标配消费股

### 9.2 基于行业生命周期的配置

**成长期行业**：
- 适合在流动性宽松期配置
- 适合在市场情绪高涨期配置
- 高风险高收益

**均衡期行业**：
- 适合长期配置
- 适合任何市场环境
- 中风险中收益

**价值期行业**：
- 适合在流动性收紧期配置
- 适合在市场情绪低迷期配置
- 低风险低收益
