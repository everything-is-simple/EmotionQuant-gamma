## 1. 文档定位

本文档是**观察留档**，不是当前主线设计文。

它保留的价值主要是：

1. 记录立花义正“逆向思维”对 EmotionQuant 的启发。
2. 说明这套方法为什么容易被误读成“把 PAS 反着做”。
3. 为未来若要研究极端情绪反向策略，保留一个起点。

当前主线中，立花义正方法**不构成** `PAS` 或 `MSS` 的正式实现来源，也不进入当前交易链路的正式契约。

---

## 2. 为什么保留，但不直接采用

立花义正材料值得保留，是因为它提醒了两件事：

1. 市场极端情绪可能对应反向机会。
2. 顺势体系之外，逆向体系也有自己的逻辑闭环。

但它不适合直接进入当前主线，原因也很明确：

1. 当前资料整理里仍有大量推测项。
2. 它更偏交易哲学和主观判断，不适合直接做日频 formal 契约。
3. 当前主线已经把 `MSS` 固定为市场级风险调节，把 `PAS` 固定为个股形态层，不宜再引入一套跨层逆向覆盖逻辑。

---

## 3. 当前仍有价值的观察

### 3.1 它强调极端情绪的重要性

立花义正思路最值得保留的一点是：

1. 市场最一致的时候，往往也最脆弱。
2. 极热和极冷状态，常常比中间状态更有信息量。
3. 真正重要的不是“和别人反着来”，而是识别一致预期何时开始失效。

### 3.2 它不是简单的“反着做 PAS”

当前更合理的理解是：

1. `PAS` 更偏结构确认后的顺势形态。
2. 立花义正更偏极端情绪下的逆向博弈。
3. 两者并不是镜像关系，更不是简单互为反信号。

### 3.3 它更接近未来研究候选，而不是当前模块依赖

若未来真的要吸收这类思想，更合理的落点可能是：

1. 极端情绪样本研究
2. `MSS` 状态机边界的解释层
3. 观察台中的专项反向策略实验

而不是直接写进当前 `PAS` registry 或 `Broker` 默认执行路径。

---

## 4. 当前已经失效或容易误导的旧口径

本次整理后，以下内容统一视为研究假设，不再视为当前主线口径：

1. 任何“极热就做空、极冷就做多”的直接规则映射。
2. 任何把立花义正方法写成 `MSS` 增强模块的伪代码。
3. 原稿中所有未被原著回收验证的入场、止损、仓位推测。
4. 基于这份材料直接安排的短期/中期/长期实施计划。

---

## 5. 当前保留结论

本文件整理后的定位固定为：

1. 它是一个有价值的逆向交易观察样本。
2. 它可以帮助解释“为什么极端情绪值得重视”。
3. 它不作为当前主线 `PAS` 或 `MSS` 的直接来源文档。
4. 它不承担任何实现计划、参数设计或测试验收职责。
