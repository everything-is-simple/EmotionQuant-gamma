## 1. 文档定位

本文档是 `PAS` 中 **BOF / Pin Bar 方向的 A 股补充参考文档**。

它保留的价值有三类：

1. 解释为什么 `BOF` 适合作为当前主线最先落地的形态。
2. 提供 `Pin Bar` 与假突破失败之间的关系说明。
3. 补充一套更贴近 A 股语境的价格行为观察视角。

本文档不是当前主线 `PAS` 的正式执行口径。当前正式设计，以 `blueprint/01-full-design/02-pas-trigger-registry-contract-annex-20260308.md` 与 `blueprint/01-full-design/06-pas-minimal-tradable-design-20260309.md` 为准。

## 2. 当前主线采用边界

当前主线从本文中主要吸收以下内容：

1. `BOF` 的反转逻辑解释
2. `Pin Bar` 作为失败突破表现形态的参考意义
3. A 股里“炸板 / 假突破前高 / 高开低走失败”这类典型场景
4. 价格行为与 K 线行为应结合位置语义理解，而不是孤立看单根蜡烛

当前主线不直接照搬以下内容：

1. 原稿中的精确 `Pin Bar` 量化阈值
2. 原稿中的完整交易系统、仓位管理和心理管理写法
3. 原稿中的固定成功率、盈亏比和目标位
4. 任何把单根 `Pin Bar` 直接当作 formal `BUY/SELL` 信号的写法

## 3. 为什么这份资料对当前 PAS 有价值

### 3.1 它解释了 BOF 的交易心理

`BOF` 不是“价格跌回去”这么简单，它本质上描述的是：

1. 市场尝试突破关键位置
2. 错误方向的追随者入场
3. 突破无法维持
4. 被套者止损，反向运动加速

这与当前主线把 `BOF` 作为第一个最小可交易形态落地，是一致的。

### 3.2 它提供了 Pin Bar 的语义补充

这份资料的长期价值，不是给出一个永恒有效的影线阈值，而是提醒：

1. `Pin Bar` 常常是失败突破的 K 线表现。
2. 但 `Pin Bar` 只有放在关键位置上才有意义。
3. 脱离关键位置单独识别 `Pin Bar`，很容易制造伪信号。

### 3.3 它更贴近 A 股实战语言

相比纯外汇语境材料，本文更容易对接 A 股里的常见现象：

1. 涨停后炸板
2. 高开低走
3. 假突破前高后回落
4. 情绪性冲高失败

这类场景对当前 `BOF` 研究尤其有解释价值。

## 4. 核心参考点

### 4.1 裸 K 线的真正含义

本文最值得保留的地方，不是“拒绝一切指标”，而是强调：

1. 价格是首要信息
2. K 线是行为结果，不是神秘符号
3. 形态必须放回结构和位置里解释

### 4.2 BOF 的核心行为链

`BOF` 可以被理解为一条连续行为链：

1. 接近关键位置
2. 尝试突破
3. 突破失败
4. 回到原区间
5. 错误方向退出，反向加速

当前主线如果要把 `BOF` 做稳，关键不是多加几个阈值，而是把这条行为链的必要节点固定清楚。

### 4.3 Pin Bar 的参考角色

`Pin Bar` 在当前主线中更适合作为：

1. `BOF` 的质量提示
2. 失败突破的可视化痕迹
3. 结构完成度的辅助解释

它不应该单独升格为当前正式 trigger。

---

## 5. A 股语境下仍有价值的观察

### 5.1 炸板是典型失败突破素材

在 A 股里，`BOF` 最自然的观察场景之一就是：

1. 盘中或当日曾表现出强突破意图
2. 但无法维持强势
3. 最终形成高位失败与回落

这使得 `BOF` 在 A 股中有明显现实基础。

### 5.2 高开低走、冲高回落有结构意义

这类 K 线如果出现在关键位置，常常意味着：

1. 上方承接不足
2. 追涨力量被消化
3. 原本被认为有效的突破开始失效

这类观察适合作为 `BOF` 解释层样本，而不是直接替代算法定义。

### 5.3 假突破前高是日线系统更易承接的形态

对于 `T+1 Open` 的日线系统而言，“假突破前高再回落”这类结构比很多更细碎的盘中波动更容易落成稳定规则，因此它对当前主线的参考价值高于大量短线 discretionary 细节。

## 6. 当前已经失效或容易误导的旧口径

本次整理后，以下内容统一降级：

1. `Pin Bar` 的固定影线比例阈值
2. `BOF` 的固定成功率与固定盈亏比
3. 原稿中的完整交易计划、仓位管理和心理管理章节
4. “某一根 K 线长得像 Pin Bar，因此直接生成信号”的写法

这些内容并非不能研究，但不属于当前主线正式语义。
