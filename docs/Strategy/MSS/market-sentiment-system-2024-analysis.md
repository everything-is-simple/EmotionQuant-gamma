## 2. 核心理论：三维观测框架

### 2.1 维度一：市场广度（Market Breadth）
- 涨跌家数比、上涨占比
- 对应MSS的"大盘系数"因子

### 2.2 维度二：情绪强度（Intensity）
- 涨停、跌停、新高、新低
- 对应MSS的"赚钱效应"和"亏钱效应"因子

### 2.3 维度三：持续性（Continuity）
- 连续涨停、连续新高
- 对应MSS的"连续性因子"

---

## 3. 七阶段情绪周期

1. **萌芽期**（<30°C）：底部信号，80%-100%仓位
2. **发酵期**（30-45°C）：温和上升，60%-80%仓位
3. **加速期**（45-60°C）：快速上升，50%-70%仓位
4. **分歧期**（60-75°C，上升/横盘）：高位震荡，40%-60%仓位
5. **高潮期**（≥75°C）：极热，20%-40%仓位
6. **扩散期**（60-75°C，下降）：高位下降，30%-50%仓位
7. **退潮期**（<60°C，下降/横盘）：持续下降，0%-20%仓位

---

## 4. 对称性设计

**赚钱效应**：涨停占比 + 新高占比 + 强势上涨占比  
**亏钱效应**：炸板率 + 跌停占比 + 强势下跌占比 + 新低占比

**温度公式**：
```
温度 = 大盘系数×0.2 + 赚钱效应×0.4 + (100-亏钱效应)×0.4
```

---

## 5. A股特色考量

- **涨跌停制度**：主板±10%、创业板±20%、ST±5%
- **T+1交易制度**：情绪判断需要"提前一天"
- **散户主导**：情绪波动大，量化系统更有价值
