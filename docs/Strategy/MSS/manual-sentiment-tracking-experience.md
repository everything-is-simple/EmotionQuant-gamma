## 1. 文档定位

本文档是 `MSS` 的**经验补充参考文档**，不是当前主线设计正文。

它保留的价值主要有三类：

1. 记录 2020-2025 年手工盯盘与手工统计的经验痕迹。
2. 为市场情绪指标、极端样本和人工复核习惯提供来源说明。
3. 解释 EmotionQuant 为什么会重视“赚钱效应/亏钱效应/持续性/仓位纪律”这些主题。

当前主线的正式设计与执行口径仍以 `blueprint/` 为准，当前固定链路是：

`Selector 初选 -> BOF 触发 -> IRS 排序 -> MSS 控仓位 -> Broker 执行`

因此，本文不再承担“如何直接生成仓位建议或交易信号”的职责。

---

## 2. 当前主线采用边界

当前主线只把本文作为以下事项的参考来源：

1. 市场情绪观测项的来源说明。
2. 极端市场样本与边界场景的复盘线索。
3. 情绪周期、炸板率、连板高度等经验变量的启发背景。
4. 人工复核为什么仍然必要的经验依据。

当前主线**不直接把本文内容当作执行规则**：

1. 不直接继承文中的固定温度阈值。
2. 不直接继承文中的固定仓位区间。
3. 不直接把文中的案例文字当回测证据。
4. 不直接把“推测项”写入当前 `MSS` 正式输入契约。

凡是需要进入当前主线的字段、公式、状态机或 Broker 覆盖语义，都应回到 `blueprint/01-full-design/04-mss-lite-contract-annex-20260308.md` 与 `blueprint/01-full-design/08-mss-minimal-tradable-design-20260309.md`。

## 3. 手工记录可能覆盖的内容

原始手工表未在本文中逐列还原，以下内容保留为**高概率覆盖项**，用于解释资料价值，不作为已验证数据字典。

### 3.1 基础观测

- 交易日期
- 核心指数涨跌幅
- 全市场上涨/下跌/平盘家数
- 市场成交额及其变化

### 3.2 情绪强弱观测

- 涨停、跌停、触板、炸板
- 连板家数与最高板高度
- 创新高、创新低家数
- 强势上涨、强势下跌家数

### 3.3 主观备注

- 当日热点板块
- 龙头股与异动股
- 特殊事件、政策扰动、黑天鹅冲击
- 当日操作与复盘结论

这里需要特别说明：凡带“推测”色彩的字段与样例，在原表未完成结构化抽取前，都只保留为经验描述，不视为正式数据事实。

## 4. 保留下来的核心经验

### 4.1 情绪周期确实可观察

长期手工记录的核心价值，不在于做出一份“完美指标表”，而在于反复观察到：

1. 市场存在由低温修复到高温分歧再到退潮的情绪循环。
2. 极端热度与极端冰点都容易对应高风险或高机会窗口。
3. 市场广度、涨停强度、炸板率和连板高度常常同步变化。

### 4.2 炸板率是高价值预警量

原经验稿中最值得保留的一点是：

1. 涨停数量多并不等于环境安全。
2. 当高热度与高炸板率并存时，情绪质量可能已经开始恶化。
3. 这类观测值得在自动化系统中保留专门入口，而不是只看单边赚钱效应。

### 4.3 仓位纪律通常比单次选股更重要

手工记录长期累积出的经验，支持保留以下结论：

1. A 股环境里，仓位放大与缩小常常比个股挑选更早决定回撤形态。
2. 市场情绪的作用更接近“风险调节器”，而不是“个股打分器”。
3. 这与当前主线中 `MSS` 只做市场级风险覆盖的定位是对齐的。

### 4.4 人工复核仍然有必要

原手工记录保留下来的另一条有效经验是：

1. 黑天鹅、监管扰动、流动性断层等事件会让日常统计短期失真。
2. 即使自动化链路已经成熟，仍应保留人工解释与异常复核空间。
3. 但人工复核应服务于边界判断，不应回到“主观拍脑袋覆盖系统”的旧路径。

## 5. 对当前自动化系统的启发

本文对当前主线真正仍有价值的地方，主要集中在以下四点：

### 5.1 保留赚钱效应与亏钱效应的对称视角

手工盯盘经验说明，单看上涨面不够，必须同步盯：

1. 涨停与炸板
2. 新高与新低
3. 强势上涨与强势下跌

这解释了为什么 `MSS` 的理论来源强调“对称观测”。

### 5.2 保留极端样本库意识

2020 年 3 月、2020 年 7 月、2021 年 2 月、2022 年 4 月这类极端阶段，即使文中具体数值尚未结构化，也仍然值得保留为：

1. 未来状态机校验样本
2. 阈值敏感性验证样本
3. 人工复核的典型边界案例

### 5.3 保留 T+1 语义下的“提前判断”

手工经验与当前主线都共享同一个现实约束：

1. `signal_date = T`
2. `execute_date = T+1`
3. 成交价语义固定为 `T+1 Open`

因此，市场情绪系统的价值不在于事后描述，而在于为次日容量和风险控制提供提前量。

### 5.4 保留人工备注，但降级为辅助手段

如果未来继续维护手工观察，建议只保留：

1. 重大事件说明
2. 异常市场氛围备注
3. 实盘操作复盘理由

不再回到大规模手工统计原始数据的旧模式。

## 6. 当前已经失效或容易误导的旧口径

本次整理后，以下内容统一降级，不再视为当前主线口径：

1. 任何未经原表回收验证的“示例日期 + 精确数值”。
2. 任何固定的温度分段与仓位百分比表。
3. “自动生成交易信号 + 人工确认”这类旧执行表述。
4. 把手工经验直接映射为当前 `MSS` 正式字段或正式公式的写法。

这些内容并非完全无用，但只能作为：

1. 历史经验
2. 研究线索
3. 后续证据抽取候选

## 7. 后续若要继续回收的证据

若后续要进一步提高本文价值，优先级应是：

1. 从原始 Excel 中抽取真实列结构，而不是继续扩写推测描述。
2. 把典型极端日期整理成结构化样本。
3. 将手工口径与当前 `L2/L3` 市场快照做逐项对照。
4. 只把经过验证的观察转化为正式测试或校准证据。

在这些工作完成前，本文定位保持为“经验补充参考”，不升级为设计文或证据文。
